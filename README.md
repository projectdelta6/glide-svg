# Glide SVG

## Installation
Use [![](https://jitpack.io/v/projectdelta6/glide-svg.svg)](https://jitpack.io/#projectdelta6/glide-svg)


```
repositories {
  ...
  maven { url 'https://jitpack.io' }
}
```
```
dependencies {
  ...
  implementation 'com.github.projectdelta6:glide-svg:4.0.2'
}
```

## Usage
```kotlin
Glide.with(this)
  .load(url)
  .into(picto_view)
```
